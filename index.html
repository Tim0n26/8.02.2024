Результат в консоли
1: Создайте массив из 10 элементов и удалите из него первые 3 элемента.
2: Создайте массив из 7 элементов и замените каждый элемент на строку “hello”.
3: Создайте массив из 6 элементов и отсортируйте его в обратном порядке.
4: Создайте массив из 4 элементов и найдите индекс элемента, который равен 3.
5: Создайте массив из 8 элементов и удалите из него все элементы, кроме первых двух.
6: Создайте массив из 9 элементов и добавьте в него элемент “world” на позицию 3.
7: Создайте массив из 12 элементов и найдите сумму всех элементов.
8: Создайте массив из 11 элементов и найдите среднее арифметическое всех элементов.
9: Создайте массив из 6 элементов и найдите максимальный элемент.
10: Создайте массив из 5 элементов и найдите минимальный элемент.
11: Создайте массив из 7 элементов и найдите количество элементов, которые больше 5.
12: Создайте массив из 10 элементов и найдите количество элементов, которые меньше 10.
13: Создайте массив из 8 элементов и объедините его с другим массивом из 4 элементов.
14: Создайте массив из 6 элементов и отфильтруйте его, оставив только элементы, которые больше 3.
